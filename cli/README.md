# Dataset Instructions

This folder is intended for storing the dataset required for seeding purposes.

## How to Download the Dataset

1. Visit the following Kaggle dataset page:
   [The Holy Quran - Kaggle Dataset](https://www.kaggle.com/datasets/zusmani/the-holy-quran)

2. Download the dataset files as `.zip` from Kaggle.

3. **Extract** the downloaded files into this `dataset` folder.

## Purpose

The files placed here will be used for database seeding and development purposes.
**Note:** The actual dataset files are not included in this repository due to licensing and size constraints.

---
**Do not commit any dataset files to version control.**
This folder is ignored for `.csv` and `.xlsx` files via `.gitignore`.
